# CRM Portal (MERN Stack)

A production-style full-stack CRM application built using **MongoDB, Express.js, React.js, and Node.js** to streamline customer relationship management, sales operations, and internal workflows.

Designed with role-based access control, analytics dashboards, and modular business entities such as Leads, Contacts, Accounts, Deals, and Tasks.

---

## Live Highlights

* Secure Login Authentication
* Role-Based Access (Admin / Seller)
* Admin Dashboard Analytics
* Seller Dashboard Analytics
* Leads Management
* Contacts Management
* Accounts Management
* Deals Pipeline Tracking
* Tasks & Deliverables Management
* AI Assistant Integration
* Responsive Modern UI

## Installation & Setup

```bash
# Clone repository
git clone https://github.com/yourusername/crm-portal-mern.git

# Move into project
cd crm-portal-mern

# Install frontend dependencies
cd frontend
npm install

# Install backend dependencies
cd ../server
npm install

# Run backend
npm run dev

# Run frontend
cd ../frontend
npm run dev
```

---

## Environment Variables

Create `.env` file inside backend:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
